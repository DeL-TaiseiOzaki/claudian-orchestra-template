# sources/（① Raw 層）

- 先方からもらった一次資料と、Inbox 由来の生データ（slack digest・code・添付など hermes capture）の curate 着地先。非 md 直置き可。
- **immutable**：リネーム・編集・削除しない（provenance 維持）。蒸留は `docs/` / `meetings/` / `references/` に別ノートとして書く。
- 詳細は [[.claude/rules/work-management.md]] を参照。
